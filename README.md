# HR System Source Code Standards

This repository defines coding standards, best practices, and source code structure guidelines
for HR systems and HRIS projects.

## Scope
- Backend (.NET / ASP.NET)
- Frontend (JavaScript / CSS)
- Database & Scripts
- Naming conventions
- Security and maintainability practices

## Notes
- Build artifacts and generated packages are excluded.
- This repository is intended as a reference standard, not a deployable application.
